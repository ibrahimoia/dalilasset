Data Preparation Guide: Asset Condition Assessment

To ensure your Excel file uploads successfully and generates accurate PDF reports, health metrics, and automated engineering recommendations, your spreadsheet must follow a specific layout structure.

📋 General Preparation Rules

  • File Format: The file must be saved as an Excel Workbook (.xlsx or .xls).
  • Sheet Structure: Place all asset data on the first tab (Sheet 1) of your workbook.
  • Header Row: Row 1 must contain the exact column headers listed below. Do not leave empty rows above your column headers.
  • Text Formatting: Avoid blank spaces inside your barcode and PIN IDs (e.g., use BC-10023 instead of BC 10023).

🏗️ Required Columns & Layout

Your Excel spreadsheet should include the following exact column headers (case-sensitive):

Column Header Name Data Type / Format Description & Purpose Example
AssetBarcode Text / Numbers Mandatory. The primary unique identifier for the specific equipment piece. Used to fetch matching photographs automatically. A-40892
PIN Text / Numbers Mandatory. The Property Identification Number. Multiple assets can share the same PIN if they belong to the same property or cluster group. PIN-QA-09
AssetDescription Text A clear description of what the asset or system item actually is. Chilled Water Pump 02
sys Text The structural engineering or mechanical system segment name. HVAC
building Text The name or specific zone designation of the building. Tower A
location Text Floor room number, sector, or exact physical placement coordinates. Basement Plant Room
AssetConditionsDescription Text Mandatory. The condition text used to calculate automatic maintenance actions. Accepted terms: Excellent, Good, Fair, Poor, Broken, New, جديد, متوسط, سيء. Fair
date-purchase Date (YYYY-MM-DD) Used by the system to compute the equipment lifecycle age context. 2018-04-15
comments Text Field notes, inspector annotations, or critical observations. Slight belt wear noted.

💡 Quick Tips for Success

⚠️ Critical Check: If an asset's AssetConditionsDescription is listed as Poor or سيء, the system automatically flags it in red with an "IMMEDIATE REPAIR / REPLACE" directive on the dashboard and generated PDFs.
📸 Photo Syncing: If you have asset photos stored in the backend, name your image file exactly after your AssetBarcode column value (e.g., A-40892.jpg or A-40892.png) so the system maps them automatically to your rows.
